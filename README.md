# CuppyBot ( DiscordBot )

[ This bot , now under development ]

## Requirement

```
1. Yarn or NPM
```

## Install

```
$ yarn
```

## Command

```
$ yarn start
$ yarn dev
```

## Bot Command

- Welcome greeting
- !hello → hello user
- !kick @xxx → kick user ( administrator )
- !ban @xxx → ban user ( administrator )
- !addUser @xxx → add user role ( only 1 user per time )
- !addCoAdmin @xxxx → add co-admin rols (only 1 user per time )
