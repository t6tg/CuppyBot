# CuppyBot ( DiscordBot )

[ This bot , now under development ]

## Requirement

```
1. Yarn or NPM
```

## Install

```
$ yarn
```

## Command

```
$ yarn start
$ yarn dev
```

## Bot Command

- Welcome greeting
- !hello → hello user
- !kick @xxx → kick user ( administrator or have permission in role)
- !ban @xxx → ban user ( administrator or have permission in role)
